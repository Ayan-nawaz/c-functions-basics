# C Functions Basics

This repository contains basic C programs that demonstrate the fundamentals of functions. These examples help beginners understand function calls, parameters, return types, and the execution flow of functions in C programming.

## 📂 Programs Included

- **decreasing.c** – Prints numbers in decreasing order using functions.
- **increasing-parametre.c** – Prints numbers in increasing order using function parameters.
- **increasingaftercall.c** – Demonstrates function execution after a function call.
- **sum-parametre.c** – Calculates the sum of two numbers using function parameters.
- **sum-return-type.c** – Calculates the sum of two numbers using a function with a return type.

## 📖 Topics Covered

- Functions in C
- Function Declaration
- Function Definition
- Function Calling
- Function Parameters
- Return Type
- Execution Flow of Functions

## 💻 Technologies Used

- C Programming Language
- GCC Compiler
- Visual Studio Code

## ▶️ How to Run

Compile and run any program using GCC:

```bash
gcc filename.c -o output
./output
```

Example:

```bash
gcc sum-return-type.c -o sum
./sum
```

## 🎯 Purpose

This repository is created for learning and practicing the basic concepts of functions in C programming. It is helpful for beginners, students, and anyone preparing for coding interviews or programming exams.
